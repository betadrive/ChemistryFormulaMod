<h1>ChemistryFormulaMod</h1>
</div>
ChemistryFormulaMod is a Fabric mod that aims to add chemical formulas to as many items and blocks in Minecraft as possible.
Our goal is to only add formulas grounded in <b>real-world data</b>.

<h2>Usage in modpacks</h2>
Feel free to use this in any modpacks. You do not need to ask for permission. You can also reference any chemical formulas in this mod as you wish.
